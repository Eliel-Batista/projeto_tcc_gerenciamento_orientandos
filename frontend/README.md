# Frontend SIAOA

## Propósito

Esta pasta contém o frontend React do SIAOA, responsável pela interface dos perfis `ORIENTADOR` e `ORIENTANDO`, dashboards, agenda e notificações.

## Estrutura proposta

- `public/`: recursos estáticos
- `src/components/`: componentes reutilizáveis
- `src/pages/`: páginas e rotas principais
- `src/services/`: integração com API REST
- `src/hooks/`: hooks personalizados
- `src/contexts/`: gerenciamento de estado global
- `src/types/`: tipos e interfaces compartilhadas
